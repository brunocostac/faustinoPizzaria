# pizzaLover

## Qual o objetivo?

Simular um aplicativo de pizzaria em iOS.


## Especificações de desenvolvimento

* Arquitetura utilizada: MVVM-C
* Desenvolvimento de telas com ViewCode
* Armazenamento local com CoreData
